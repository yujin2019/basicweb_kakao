가장 많이 사용되는 IntelliJ IDEA 단축키 15개


연번 단축키 설명 비고
1 Alt + Enter Import 되지 않은 Class import 하기
2 F2 다음 오류, 경고, 제안으로 점프
3 Alt + 1 프로젝트 창 열고 닫기
4 Esc 에디터에 다시 포커스 위치
5 Ctrl + e 최근 실행 파일 확인
6 Ctrl + b 선언한 위치로 이동
7 Alt + F7 검색기능
8 Ctrl + Ctrl 어떤항목이든 실행
9 Ctrl + w / Shift + Ctrl + w 근처의 코드 선택 영역 확대 혹은 축소
10 Ctrl + / 해당 줄의 코드 주석 변환
11 Shift + Ctrl + Enter 현재 구문 완성
12 Ctrl + Alt + l 현재 파일의 서식을 지정
13 Shift + Ctrl + Alt + t 사용가능한 리팩토링 옵션이 표시
14 Ctrl + Shift + a Find Action(단축키 찾기)
15 Shift + Shift 전체검색
